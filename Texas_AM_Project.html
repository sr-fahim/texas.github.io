About the Project

Understanding a plantâ€™s root system architecture (RSA) is crucial for a variety of plant science problem domains including sustainability and climate adaptation. Minirhizotron (MR) technology is a widely-used approach for phenotyping RSA non-destructively by capturing root imagery over time.

Dataset Overview

This dataset includes over 72K RGB root images from six species: cotton, papaya, peanut, sesame, sunflower, and switchgrass. Images span varied conditions, including root age, structures, soil types, and depths. Each image is labeled to support research in semantic segmentation tasks using deep learning.

Applications

The dataset facilitates automatic root segmentation and research of RSA with advanced algorithms in machine learning and image analysis.
